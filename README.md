# AI Interview Copilot

AI-powered mock interview app. Upload a PDF resume, pick a job role, and get personalized interview questions generated by Google Gemini based on your projects, skills, and experience.

## Tech Stack

- **Frontend:** React 19, TypeScript, Vite, Tailwind CSS, React Router
- **Backend:** FastAPI, Python 3.12, Google Gemini (`gemini-2.5-flash`), pypdf

## Prerequisites

- Node.js 18+
- Python 3.12+
- A [Google AI Studio](https://aistudio.google.com/) API key for Gemini

## Setup

### 1. Backend

```bash
cd backend
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env            # then add your GEMINI_API_KEY
uvicorn main:app --reload --port 8000
```

The API runs at `http://localhost:8000`. Health check: `GET /`.

### 2. Frontend

```bash
cd frontend
npm install
npm run dev
```

The app runs at `http://localhost:5173`.

## Environment Variables

| Variable | Location | Description |
|----------|----------|-------------|
| `GEMINI_API_KEY` | `backend/.env` | Google Gemini API key (required) |

Copy `backend/.env.example` to `backend/.env` and fill in your key. **Never commit `.env`.**

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/` | Health check |
| POST | `/upload-resume` | Upload PDF resume → parsed sections |
| POST | `/start-interview` | Generate interview questions from sections + role |

## User Flow
